Kebijakan Privasi Saku

Terakhir diperbarui: Juni 2026

Catatan: Saku adalah aplikasi simulasi mobile banking untuk tujuan pembelajaran dan portfolio. Aplikasi ini bukan layanan keuangan nyata.

1. Tentang Aplikasi

Saku adalah aplikasi simulasi mobile banking berbasis Android. Seluruh saldo, nomor rekening, transaksi, transfer, top up, dan aktivitas lain di dalam aplikasi bersifat simulasi dan tidak merepresentasikan uang asli, rekening bank asli, atau layanan keuangan resmi.

2. Data yang Dikumpulkan dan Disimpan

Aplikasi dapat menyimpan data berikut untuk menjalankan fitur aplikasi:

  • Nama pengguna
  • Email pengguna
  • UID akun Firebase
  • Nomor rekening simulasi
  • Saldo simulasi
  • Riwayat transaksi simulasi
  • Rekening tujuan tersimpan
  • Firebase Cloud Messaging token untuk kebutuhan notifikasi
  • Waktu pembuatan atau pembaruan data tertentu

3. Penggunaan Data

Data digunakan untuk:

  • Membuat dan mengelola akun pengguna
  • Menampilkan profil pengguna
  • Menampilkan saldo simulasi
  • Memproses transfer simulasi antar akun Saku
  • Memproses top up saldo simulasi
  • Menampilkan riwayat dan detail transaksi
  • Menyimpan rekening tujuan yang pernah digunakan
  • Mendukung fitur notifikasi aplikasi

4. Penyimpanan Data

Saku menggunakan Firebase Authentication untuk autentikasi akun dan Cloud Firestore untuk menyimpan data aplikasi seperti profil, saldo simulasi, riwayat transaksi, dan rekening tersimpan.

5. Pembagian Data kepada Pihak Ketiga

Saku tidak menjual data pengguna kepada pihak ketiga. Data dapat diproses melalui layanan Firebase dari Google untuk mendukung autentikasi, database, dan notifikasi aplikasi.

6. Notifikasi

Aplikasi dapat menyimpan token Firebase Cloud Messaging untuk kebutuhan notifikasi. Aplikasi juga dapat menampilkan notifikasi lokal setelah transaksi tertentu berhasil dilakukan.

7. Hapus Akun dan Data

Pengguna dapat menghapus akun melalui aplikasi pada menu Profile > Hapus Akun. Setelah akun dihapus, data profil, riwayat transaksi milik pengguna, rekening tersimpan, dan akun Firebase Authentication akan dihapus dari sistem aplikasi.

Informasi lebih lanjut dapat dibaca di halaman: Panduan Penghapusan Akun dan Data.

8. Keamanan

Saku menggunakan Firebase Authentication dan Firestore Security Rules untuk membantu membatasi akses data. Namun, aplikasi ini tetap merupakan aplikasi simulasi dan portfolio, bukan sistem finansial production.

9. Bukan Layanan Keuangan

Saku bukan aplikasi bank, dompet digital, pinjaman, investasi, pembayaran, atau layanan keuangan resmi. Aplikasi ini tidak terhubung dengan bank, payment gateway, atau institusi keuangan mana pun.
