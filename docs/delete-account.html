Penghapusan Akun dan Data Saku

Terakhir diperbarui: Juni 2026

Penting: Penghapusan akun bersifat permanen. Data yang sudah dihapus tidak dapat dikembalikan.

1. Cara Menghapus Akun dari Aplikasi

Pengguna dapat menghapus akun langsung dari aplikasi Saku dengan langkah berikut:

  1. Buka aplikasi Saku
  2. Login ke akun yang ingin dihapus
  3. Buka menu Profile
  4. Pilih Hapus Akun
  5. Masukkan password akun untuk konfirmasi
  6. Konfirmasi penghapusan akun

2. Data yang Akan Dihapus

Ketika akun dihapus dari aplikasi, data berikut akan dihapus:

  • Data profil pengguna
  • Nama dan email yang tersimpan di database aplikasi
  • Nomor rekening simulasi
  • Saldo simulasi
  • Riwayat transaksi milik pengguna
  • Rekening tujuan tersimpan
  • Akun Firebase Authentication

3. Data yang Mungkin Tetap Tersimpan Sementara

Dalam kondisi tertentu, salinan data teknis dapat tetap tersimpan sementara pada sistem Firebase atau backup layanan pihak ketiga sesuai kebijakan penyedia layanan. Data tersebut tidak digunakan untuk layanan keuangan nyata.

4. Jika Tidak Bisa Login

Jika pengguna tidak dapat login tetapi ingin meminta penghapusan akun/data, pengguna dapat menghubungi developer melalui email: hendrysetiawan97@gmail.com

Sertakan informasi berikut:

  • Subjek email: Request Delete Account - Saku
  • Email akun Saku yang ingin dihapus
  • Alasan singkat permintaan penghapusan, jika diperlukan

5. Waktu Pemrosesan Permintaan Manual

Permintaan penghapusan akun/data melalui email akan diproses secara manual dalam waktu yang wajar setelah kepemilikan akun dapat diverifikasi.
