# Saku – Mobile Banking Simulation App

Saku is a Android mobile banking simulation application built using Kotlin and Jetpack Compose.  
This project was developed to explore modern Android development practices such as MVVM Architecture, Firebase integration, Dependency Injection, and State Management.

## Features

- User Authentication (Login & Register)
- Auto Login Session
- Dashboard with User Data
- Balance Management
- Auto Generated Account Number
- Money Transfer Between Users
- Saved Recipient / Saved Account Number
- Top Up Simulation
- Transaction History
- Transaction Detail Screen
- Transfer Receipt Screen
- Top Up Receipt Support
- Profile Screen
- Privacy Policy Screen
- Delete Account Flow
- Local Notification After Transfer
- Open Transaction Detail from Notification
- Firebase Firestore Integration
- Firebase Cloud Messaging Token Sync
- Firestore Security Rules
- Modern UI with Jetpack Compose
- MVVM Architecture
- Repository Pattern
- Hilt Dependency Injection
- Unit Testing
- Compose UI Testing

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- MVVM Architecture
- Hilt Dependency Injection
- Firebase Authentication
- Cloud Firestore
- Kotlin Coroutines
- StateFlow
- Navigation Compose
